# Publicar no GitHub sem terminal (somente navegador)

## Objetivo

Subir o projeto para `https://github.com/jozivam/RepoDelivery` sem usar Linux local e sem comandos.

## Passo a passo (somente cliques)

1. Abra `https://github.com/jozivam/RepoDelivery`.
2. Clique em **Add file**.
3. Clique em **Upload files**.
4. Envie os arquivos do projeto:
   - `app/main.py`
   - `tests/test_api.py`
   - `requirements.txt`
   - `README.md`
   - `GUIA_PUBLICACAO_NAVEGADOR.md`
5. Em *Commit changes*, escreva: `feat: mvp inicial`.
6. Clique em **Commit changes**.
